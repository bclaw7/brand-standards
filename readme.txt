Create and manage brand guidelines with pre-built patterns and templates.

== Description ==

Brand Standards plugin allows you to quickly create and manage brand guidelines using pre-built patterns and templates.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/brand-standards` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
